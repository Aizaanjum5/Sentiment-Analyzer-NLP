# University Review Sentiment Analyzer

NLP project that classifies university and course reviews into **positive**, **neutral**, or **negative** sentiment using traditional ML models and an LSTM-based deep learning model.

## Features

- Logistic Regression with TF-IDF features.
- Naive Bayes with Bag-of-Words features.
- LSTM model using word sequences.
- Streamlit web app to compare model predictions on the same review.
